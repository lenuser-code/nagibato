# initScroll()

ラップされているSelectオブジェクトのindexの値を元に, this.scroll と this.offset の値を初期化する.

具体的には次のように決める.

  1. indexの値がviewCount未満なら, scroll = 0, offset = index でよい.
  2. そうでないとき, 先に offset = viewCount - 1 を決定してしまい,

それから scroll = index - offset とすればよい.
