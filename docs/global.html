    (constant) LocalStorageInfo :Object

    localStorageの読み書きを行うオブジェクト. 読み書きした情報の保持も行う. そのため, プログラムがlocalStorageを 使用しているか否かは, このオブジェクトのisUsed()メソッドで判別できる.

    Properties:
    Name Type Description
    config Array.<(string|number|boolean)> | null

    設定情報のうちlocalStorageに保存するものの配列. 現在のバージョンでは,

    1. メインカードのID (string) 2, コンボ成立ルールのバージョン (number)
    2. スキル持ちカードのマーカー設定 (boolean) がこの順番に格納されている. ロードする前, および, localStorageを利用しない設定のときはnull
    deckInfo Array.<string> | null

    デッキに含まれる各カードのIDを並べた配列. ロードする前, および, localStorageを利用しない設定のときはnull
