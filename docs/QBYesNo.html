Class

QBYesNo

QBYesNo(msg, minWaitopt, seNameopt)

ユーザーに入力を要求する会話ダイアログのクラス. 見た目はQBTalkと同じだが, ユーザーがAまたはSのキーを押すまで消えない. また, Aを押した場合はtrue, Sを押した場合はfalseをresult要素に保存する.

Constructor

# new QBYesNo(msg, minWaitopt, seNameopt)

指定されたメッセージを表示するインスタンスを作る. 完全にメッセージが表示されてから入力受付を開始するまでの待ち時間も 合わせて設定する. また, 必要ならAキーが押された場合にSEを鳴らすことができる.
