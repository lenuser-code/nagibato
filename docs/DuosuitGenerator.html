# paintCost(ctx, x, y, marks, n)

指定された座標にカードの左上端があるものとしてカードのコストを描画する. 他の描画メソッドとは違い, コストはキャッシュ画像には書き込まず, PrismaticCardのpaintメソッドから毎フレーム呼び出される (さもなければ キャッシュする画像の枚数がコストの種類数だけ倍増してしまう). そのため, 基点となる(x,y)の情報が必要になる.
